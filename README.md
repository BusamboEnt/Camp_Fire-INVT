# Campsite Commander

Campsite Commander is a full-screen mobile application concept designed to help outdoor enthusiasts manage their camping gear and food supplies. Built using structural programming principles like parallel arrays, loops, and custom navigation flows, the app serves as a robust prototype for an organized packing checklist.

---

## Features

* Full-Screen Experience: Optimized UI for maximum view space and focus.
* Lazy Loading Splash Screen: A 3-second introductory display featuring asset scaling.
* Dark Mode Theme: A default nature-inspired dark color palette (#492E26).
* Inventory Hub: A centralized dashboard tracking real-time packing statistics.
* Detailed Checklist View: Categorized lists displaying item details, quantities, and custom packing tips.
* Input Validation: Robust error handling preventing empty fields, bad data types, or inventory overflow.

---

## Technical Specifications

### Architecture
The core architecture utilizes parallel arrays to maintain data integrity across distinct attributes without relying on complex object-oriented data models.


### Algorithms and Logic
* Iterative Summation: A standard loop traverses the item quantities array to compute total assets packed for the main dashboard.
* Sequential Navigation: Custom state-routing handles transitions between the Splash Screen, Main Screen, and Detailed Checklist Screen.
* Form Sanitation: Validates user input to prevent empty fields and handle numerical parse exceptions.
